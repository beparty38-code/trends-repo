What is it?

코딩 에이전트를 위한 완전한 소프트웨어 개발 방법론이자 스킬 프레임워크입니다. 에이전트가 코드를 바로 작성하지 않고, 먼저 목표를 명확히 한 뒤 체계적으로 개발하도록 가이드합니다.

Core Workflow (7 Steps)

에이전트가 작업을 시작하면 자동으로 이 워크플로우를 따릅니다:

  1. brainstorming — 소크라테스식 설계 정제. 바로 코딩하지 않고 목표부터 정리
  2. using-git-worktrees — 격리된 개발 워크스페이스 생성 (메인 브랜치 안전)
  3. writing-plans — 2~5분 단위 작업 계획 수립 ("주니어 엔지니어가 이해할 수준")
  4. subagent-driven-development — 하위 에이전트 병렬 실행 + 2단계 검토
  5. test-driven-development — RED → GREEN → REFACTOR 순환 강제
  6. requesting-code-review — 계획 대비 코드 검토
  7. finishing-a-development-branch — 병합/PR 결정
