What is it?

각 분야 전문가 역할을 하는 AI 에이전트 페르소나 컬렉션입니다. 마크다운 파일로 정의되며, Claude Code, Cursor, Copilot 등 다양한 도구에 설치해 사용합니다. 총 232개 전문 에이전트가 16개 부서에 걸쳐 있습니다.

- [책임 2]

[단계별 프로세스]

[측정 가능한 결과]

Installation

Native App (macOS)

brew install --cask msitarzewski/agency-agents/agency-agents

Claude Code

./scripts/install.sh --tool claude-code
# 또는 직접 복사
cp engineering/*.md ~/.claude/agents/

Interactive

./scripts/install.sh                    # 대화형 마법사
./scripts/install.sh --list teams       # 팀 목록 확인
./scripts/install.sh --tool cursor      # Cursor용
./scripts/install.sh --tool copilot     # Copilot용
./scripts/install.sh --tool gemini-cli  # Gemini용

Learn Points

  • 효과적인 시스템 프롬프트 작성 기법 (100+ 실전 예시)
  • 페르소나 설계 — 성격, 워크플로우, 톤 정의 패턴
  • 에이전트 역할 분리 — 부서별 전문화 전략
  • 마크다운 기반 선언적 에이전트 정의 방식
  • Multi-tool 호환 설치 스크립트 설계
