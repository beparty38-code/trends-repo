ChromeDevTools/chrome-devtools-mcp

45.6k stars TypeScript AI Infra

What is it?

AI 코딩 에이전트(Claude, Cursor, Copilot 등)가 라이브 Chrome 브라우저를 제어하고 검사할 수 있게 하는 MCP(Model Context Protocol) 서버입니다.

Project Structure

chrome-devtools-mcp/
├── src/                    # 핵심 소스 코드
├── docs/                   # 도구 참조, CLI, 트러블슈팅
├── skills/                 # AI 에이전트용 스킬 정의
├── tests/                  # 테스트 스위트
├── .claude-plugin/         # Claude 플러그인 설정
├── .cursor-plugin/         # Cursor 플러그인 설정
├── .gemini/                # Gemini 설정
├── rollup.config.mjs       # 번들링 설정
├── puppeteer.config.cjs    # Puppeteer 설정
└── package.json

Installation

Claude Code

claude mcp add chrome-devtools --scope user npx chrome-devtools-mcp@latest

Universal (MCP JSON config)

{
  "mcpServers": {
    "chrome-devtools": {
      "command": "npx",
      "args": ["-y", "chrome-devtools-mcp@latest"]
    }
  }
}

Slim Mode (경량)

npx chrome-devtools-mcp@latest --slim --headless
